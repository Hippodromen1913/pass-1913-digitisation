# Digitisation of "Pass" – Hippodromen New Year Revue (Malmö, 1913)

This project is part of the course *Digitising Cultural Heritage Material* at the University of Borås.

## Project Description

The aim of this project is to digitise a printed booklet from the New Year revue *Pass*, performed at Hippodromen in Malmö in 1913. The booklet mainly consists of songs and dialogue, along with a limited number of images.

The material was digitised from a physical copy held at Malmö City Library. The project aims to make the booklet available in a structured digital format while also adding contextual information about the revue and its historical setting.

## Objectives

- Digitise the complete booklet through scanning
- Transcribe the text using OCR and manual correction
- Encode the text using TEI P5 XML
- Create descriptive, structural, and technical metadata
- Add contextual information about the performance
- Publish the material digitally

## Methods and Tools

- Scanning of printed material
- OCR using Google Docs, followed by manual correction and verification
- TEI P5 XML encoding
- GitHub for version control and publication
- Google Drive for storage of original image files

## Metadata

The project uses descriptive, structural, and technical metadata.

Descriptive metadata includes information about the revue, publication, and historical context. Structural metadata is mainly handled through the TEI encoding. Technical metadata documents parts of the digitisation process, such as file formats, image resolution, OCR processing, and storage of the original files.

## Digitisation Specifications

Original image files were scanned as high-resolution TIFF files in 600 dpi RGB colour. The TIFF files function as preservation masters, while compressed GIF versions are used as access copies in the GitHub repository due to file size limitations.

## Notes

The material is primarily textual, which has influenced the focus on transcription and encoding, although it also contains a small number of images that require separate handling and metadata. OCR provided a useful starting point for transcription but still required manual correction. These challenges are discussed further in the project documentation and final report.
